
# Neomorphic-PortfolioTemplate
This is a OpenSource Portfolio Template Based on Neomorphic Theme Made with pure HTML, CSS(SCSS), And A bit of Java script

- Rich, multi-step animations 
- Fullscreen mode
- Aesthetic and modern UI
